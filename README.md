This repository contains the Luminex assay data processing pipelines used in the publication:

"Dysregulation of the TGF-β Superfamily Across HFpEF and PAH: Distinct Mechanisms, Shared Therapeutic Potential"

Carolina Janssen-Telders, Eszter Nóra Tóth, Tobias Neumann, Clarissa Becher, Jan Willem Duitman, Tamara Dekker, Aida Llucià-Valldeperas, Jessie van Wezenbeek, Lucas Celant, Joris R. de Groot, Harm Jan Bogaard, M. Louis Handoko, Marie José Goumans, Frances S. de Man

Two Luminex experiments were performed. The corresponding data processing was conducted separately, and the pipelines can be found in the 1st_Run and 2nd_Run directories.

The same packages and parameter settings were used for both pipelines.
